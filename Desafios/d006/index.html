Tags em HTML

HTML é uma linguagem de código - não de programação - usada para desenvolver a estrutura de um site. Nele, você indica ao navegador, o qual lerá o arquivo .html, o que cada texto, espaço ou mídia é dando um sentido a eles através das tags. A seguir, mostro-te alguns exemplos de tags:

  • <p>
  • <h1>
  • <h2>
  • <hr>
  • <br>
  • <img src="">
  • <a href="">
