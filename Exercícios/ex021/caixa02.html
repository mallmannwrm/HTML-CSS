Grouping Tags

Auxiliam os navegadores a compreender a estrutura do site

<header> é a tag de cabeçalho do body
Link Link Link
<main> é a tag de parte principal do body.
<section> é uma seção, geralmente usada dentro do <main>
<article> é usado para delimitar artigos.
<aside> significa "periférico", ou seja, faz parte de determinado bloco de informações, mas não diretamente.
<nav> é usada para menus.

<footer> é a tag de rodapé do body
