Tag "code" e "pre"

Code faz com que um texto tenha uma fonte monoespaçada, facilitando a leitura. Exemplo:
arquivo.pdf

Já pre faz com que qualquer espaço presente na escrita html seja levado para o site. Exemplo:

    Olá, tudo bem?
        Tudo. E você?
            Bem.
                Que bom!

Tags para citações

Adiciona aspas em textos com sentido exclusivo de citação. Em outros contextos, vale colocar as aspas normalmente.

<q> é a tag para citações rápidas.
O pior crime ao ser humano é reduzi-lo a objeto

No caso de citações completas, usa-se <blockcote>. Adicionando "cite" a ela é possível colocar o endereço web fonte da citação para que o sistema interno do site tenha esta informação.

A educação é a arma mais poderosa que você pode usar para mudar o mundo.

Indicação de siglas

A tag <abbr> permite adicionar o nome completo que aparece ao passar o cursor sobre a sigla.

IBGE
