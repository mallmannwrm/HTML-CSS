Ordened lists

Listas em que há uma enumeração e ordem dos itens. O parâmetro type dentro da tag <ol> permite mudar o tipo de numeração.

  1. Acordar
  2. Verificar o celular
  3. Trocar de roupa
  4. Fazer skincare e escovar os dentes
  5. Tomar café da manhã

Listas não ordenadas

Unordened lists

Listas em que a ordem dos itens não importa. O parâmetro type dentro da tag <ul> define o tipo de marcação

  • Lápis
  • Papel A4
  • Caneta BIC
  • Garrafa D'água
  • Borracha

Mistura de listas

  1. Sistema Windows
    • Tamanho maior
    • Mais acessível aos programas
    • Polêmica de privacidade
  2. Sistema Linux
    • Código aberto
    • Mais difícil ao usuário casual
    • Tamanho menor
