Ligações/links

Ao interagir com um hyperlink, você pode ser redirecionado para outra página através de um URL ou fazer download de um arquivo.

Âncoras/links externos

Exemplo de acesso a site pela tag <a>. Ainda, os parâmetros target="_blank" e rel="external" indicam, respectivamente, que o site deve ser aberto em uma nova aba e que ele é externo, ou seja, não está no banco de dados do site atual.

Já o parâmetro href="[URL]" é obrigatório para inserir o URL do site que se deseja direcionar.

Links internos

Liga paǵinas que estão no servidor junto com o site principal. Acesse a segunda página aqui. Nestes casos, o ideal é que a URL abra na página principal. Ainda, não é necessário que a URL inteira seja digitada.

Downloads

O processo é o mesmo, mas agora é preferível que os arquivos estejam em uma pasta separada. Ainda, é necessário usar o parâmetro type="[específico de cada tipo de arquivo]"
